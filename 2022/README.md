Install neovim
!(https://github.com/neovim/neovim/releases/tag/v0.6.1)[]

2) Extract: tar xzvf nvim-macos.tar.gz
3) Run ./nvim-osx64/bin/nvim

Install neovim extensions
1) open ```~/.config/nvim/init.vim```
2) run :PlugInstall
3) run :CocInstall
